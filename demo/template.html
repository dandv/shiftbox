An <x-shiftbox> is similar to hamburger panel except it can shift in any direction.

The shiftbox requires a section element and an aside element. The section sits on top of the aside element and is shifted to the side when the "open" attribute is applied or the "toggle()" method is called. By setting the shift attribute you can control which direction the section element moves to.

Basic usage

Markup

<x-shiftbox>

<section>

  <h1>Content</h1>

  <ul>

    <li>first</li>

    <li>second</li>

    <li>third hello hello hello</li>

  </ul>

</section>

<aside>

  <h1>Sub Content</h1>

  <ul>

    <li>News</li>

    <li>Blog</li>

    <li>About</li>

  </ul>

</aside>

</x-shiftbox>

JavaScript

// assume shiftbox is already defined as their respective DOM elements

// go back a slide

shiftbox.addEventListener("closed", function(){

   console.log('closed')

});

shiftbox.addEventListener("opened", function(){

   console.log('opened')

});

shiftbox.toggle();
