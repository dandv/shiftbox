An <x-shiftbox> is similar to hamburger panel.

However, the main difference is that <x-shiftbox> treats the slides as a coherent filmstrip instead of as independent cards, so transitioning from one slide to another displays all slides in between during the animation.

Basic usage

Markup

<x-shiftbox>

<section>

  <h1>Content</h1>

  <ul>

    <li>first</li>

    <li>second</li>

    <li>third hello hello hello</li>

  </ul>

</section>

<aside>

  <h1>Sub Content</h1>

  <ul>

    <li>News</li>

    <li>Blog</li>

    <li>About</li>

  </ul>

</aside>

</x-shiftbox>

JavaScript

// assume shiftbox is already defined as their respective DOM elements



// go back a slide

shiftbox.addEventListener("closed", function(){

   console.log('closed')

});



shiftbox.addEventListener("opened", function(){

   console.log('opened')

});



shiftbox.toggle();
